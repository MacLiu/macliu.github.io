Projects

State Class Alert

Get a notification when a class you want opens up!

During dead week of Spring 2017, I took it upon myself to reverse-engineer Iowa State University's Classes API and created a mobile and web app on top of it. The app allows students to choose from a list of the university's classes and sections they are trying to get into then alerts them when an open seat is available. Try it out with the link below!

http://stateclassalert.com

MemeMarket

MemeMarket is a web application created to visualize the popularity of image templates on reddit over time. Developed the API utilizing Node.js, Express.js, and MongoDB as part of HackISU 2016.

eHireMe

eHireMe is a mobile application that intelligently matches employers and potential new hires by experience, location, and interests. A match is created when both the employers and potentional candidates swipe right on eachother. Then, the users are brought into a chat to schedule an interview.

SplashTalk

Splash Talk is an Android application that brings texts into life by adding sound effects based off of the mood of the text. Developed an original algorithm using a Naive Bayes Classifier and the "bag of words" model to perform Sentiment Analysis.

Expo

Expo is a mobile app designed specifically for trading between college students. It's a simple way to list your items for sale and buy things that others are selling. All transactions are handled within the app by utilizing the Venmo API. This app won the HackISU Fall 2015 Venmo API prize among all entries!

FaceRate

For CodeDay 2014, I developed an iOS app that rates users based off their facial features' deviation from the Golden Ratio. This app uses a facial recognition API (Face++) that extracted landmarks and facial key-points from an image. Utilizing the data, I implemented my own algorithm to calculate the ratios and symmetry of the face and determined the deviation from the Golden Ratio. The app won the mobile app division among all entries.

MyDining

For HackISU 2014, we reverse-engineered the API of ISU Webfood, Iowa State University's Flash-based online food ordering system, and built an iOS mobile application on top of it using Swift. We developed a fully functioning app during the hackathon and ordered breakfast Sunday morning using our app! The application took 3rd place out of over 150 people!

SplitPay

For HackMidWest 2014, we created SplitPay, an iOS & Android application that allows a user to create a fund, which others can contribute to. Once enough money is generated for the fund, the app gives all the money to the creator of the fund. Utilized the Simplfy API for payment acceptance and transactions. SplitPay took runner-up at HackMidwest 2014.
