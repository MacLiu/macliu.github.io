Projects

State Class Alert

Get a notification when a class you want opens up!

During dead week, I took it upon myself to reverse-engineer my university's classes API and create a mobile app on top of it. This app allows students to choose a list of classes and sections that they are trying to get into and alerts them when an open seat is available!

EHireMe

EHireMe is a mobile application that matches employers and potential new hires. When the employer and potentional new hires swipe right on eachother, a match is created and the two users are brought into a chat to setup an interview time.

SplashTalk

Splash Talk is an Android application that brings texts to life by adding sound effects based off of the mood of the text. This is done using Naive Bayes Classifier for NLP to perform Sentiment Analysis on the text to determine the mood.

Expo

Expo is a mobile app designed specifically for trading between college students. It's a simple way to list your stuff for sale and buy things that others are selling. This app was made for the Fall 2015 HackISU, and we were the Venmo API winners!

MyDining

For HackISU 2014, we reverse-engineered our school's flash-based online food ordering system, ISU Webfood, and built an iOS mobile application on top of it using Swift. We manages to have the whole application up and running by Sunday morning and even placed an order in their system. The project took 3rd place of over 150 people!
