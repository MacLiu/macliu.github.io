Projects

State Class Alert

Get a notification when a class you want opens up!

During dead week of Spring 2017, I took it upon myself to reverse-engineer my university's classes API and create a mobile and web app on top of it. The app allows students to choose from a list of the university's classes and a sections that they are trying to get into and alerts them when an open seat is available!

http://stateclassalert.com

EHireMe

EHireMe is a mobile application that matches employers and potential new hires. When the employer and potentional new hires swipe right on eachother, a match is created and the two users are brought into a chat to setup an interview time.

SplashTalk

Splash Talk is an Android application that brings texts to life by adding sound effects based off of the mood of the text. This is done using Naive Bayes Classifier for NLP to perform Sentiment Analysis on the text to determine the mood.

Check it out in the Play Store!

Expo

Expo is a mobile app designed specifically for trading between college students. It's a simple way to list your stuff for sale and buy things that others are selling. All transactions are handled within the app with the Venmo API.This app was made for the Fall 2015 HackISU, and won the Venmo API prize!

MyDining

For HackISU 2014, we reverse-engineered our school's flash-based online food ordering system's, ISU Webfood, API and built an iOS mobile application on top of it using Swift. We developed a fully-functioning app during the hackathon and ordered breakfast Sunday morning with the app! The application took 3rd place out of over 150 people!
